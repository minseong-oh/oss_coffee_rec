# 커피 원두 추천 앱

오픈소스소프트웨어실습 기말과제 - Streamlit + FastAPI + Docker + EC2 통합 추천 앱.

사용자의 커피 취향(산미·바디·로스팅·향·단맛·카페인·추출방식)을 입력받아
원두 데이터베이스(12종)와 가중치 점수화 매칭으로 TOP3 원두를 추천한다.

## 점수 기준 (100점 만점)

| 항목 | 배점 |
|---|---|
| 산미 매칭 | 20 |
| 바디 매칭 | 20 |
| 향 프로파일 매칭 | 20 |
| 로스팅 매칭 | 15 |
| 단맛 매칭 | 10 |
| 카페인 매칭 | 10 |
| 추출 방식 매칭 | 5 |
